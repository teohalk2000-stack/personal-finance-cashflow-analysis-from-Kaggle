## Dataset
- Period: 2020–2024  
- Includes:
  - Date  
  - Category  
  - Amount  
  - Type (Income / Expense)

Source: https://www.kaggle.com/datasets/ramyapintchy/personal-finance-data

## Process
- Data cleaning (missing values, inconsistencies)  
- Category grouping  
- Duplicate removal  
- Data normalization  

## Analysis
- Expense ranking by category  
- Monthly expense trends  
- Savings rate calculation  
- Statistical testing (t-test)  

## Key Insights
- Highest expenses: August  
- Main spending category: Living & Entertainment  
- Highest savings rate: October  
- Some months show negative savings
